MLA Format

  • 12pt Times New Roman
  • Double-spaced, 1-inch margins
  • Header with Last Name, Page #
  • In-text citations: (Author Page)
  • Works Cited Page: Author. Title. Publisher, Year.

APA Format

  • 12pt font (Times New Roman preferred)
  • Running Head on each page
  • Title page, abstract, main body, and references
  • In-text citation: (Author, Year)
  • Reference Page: Author, A. (Year). Title. Publisher.

Helpful Software

  • PERRLA: Word add-in for formatting citations
  • Grammarly: Checks citations and grammar
  • Zotero or EasyBib for bibliography generation
